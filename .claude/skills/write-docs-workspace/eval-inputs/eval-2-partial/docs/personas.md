# 페르소나

## 주차 담당자
- **역할**: 주차장 운영 관리
- **기술 수준**: 중
- **주요 목표**: 빈 자리 현황 파악, 예약 승인/거절
- **주요 시나리오**:
  1. 하루 예약 현황 확인
  2. 긴급 예약 처리
  3. 월별 통계 확인

## 일반 직원
- **역할**: 주차 예약 이용자
- **기술 수준**: 하
- **주요 목표**: 출근 전 주차 자리 예약
- **주요 시나리오**:
  1. 주차 자리 사전 예약
  2. 예약 취소
  3. 예약 내역 확인
