# 유저 시나리오: 주차 담당자

## 하루 일과
- **08:00**: 대시보드에 접속해 당일 예약 건수, 빈 자리 수, 취소 건수를 확인한다.
- **09:00**: 특정 직원이 예약을 요청하면 담당자가 대신 예약을 생성한다.
- **10:30**: 장기 수리로 비워야 하는 구역을 비활성화 처리한다.
- **17:00**: 월별 예약 통계를 조회해 팀장에게 보고한다.

## 미결 결정 사항
1. 담당자가 다른 직원 예약을 강제 취소할 때 취소 사유를 필수 입력해야 하는가?
2. 비활성화된 자리에 이미 잡힌 예약은 자동 취소되는가?
3. 월별 통계 리포트는 앱 내 뷰로 제공하는가, CSV 다운로드만 지원하는가?

# 유저 시나리오: 일반 직원

## 하루 일과
- **전날 22:00**: 다음 날 사용할 주차 자리를 검색하고 예약한다.
- **당일 08:50**: 예약 목록에서 예약 번호와 자리 위치를 확인하고 주차장으로 향한다.
- **13:00**: 오후 일정이 생겨 예약 시간을 수정한다.
- **예외**: 재택근무로 바뀌면 예약을 취소하고 자리를 반납한다.

## 미결 결정 사항
1. 당일 예약도 허용하는가, 아니면 전날 자정까지만 예약 가능한가?
2. 예약 변경은 시작 시각 몇 분 전까지 허용하는가?
3. 동일 직원이 같은 날 복수 자리 예약을 허용하는가?
