# 페르소나 정의

## 1. 주차 담당자 (Parking Manager)

- **역할**: 주차장 운영 총괄 담당자 (총무팀)
- **기술 수준**: 사내 시스템 사용 경험 있음. 개발 지식 없음.
- **목표**: 주차 구역 현황 파악, 예약 승인·거절·취소, 구역 비활성화, 이용 통계 조회
- **권한**: 전체 예약 조회, 예약 강제 취소, 주차 구역 등록·수정·비활성화

## 2. 일반 직원 (Employee)

- **역할**: 차량을 운행하는 일반 임직원
- **기술 수준**: 스마트폰·PC 인터넷 사용 능숙
- **목표**: 출근 전 빈 주차 자리 예약, 예약 상태 확인, 예약 취소
- **권한**: 본인 예약 생성·조회·취소, 예약 가능 구역 조회
